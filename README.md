# ♿ Formação Sem Barreiras

Projeto acadêmico desenvolvido durante o curso técnico no **SENAI**, com foco em acessibilidade e inclusão profissional para pessoas com deficiência auditiva.

O sistema foi desenvolvido para auxiliar usuários na preparação para entrevistas de emprego, oferecendo recursos acessíveis, capacitação profissional e simulações práticas.

---

## ✨ Sobre o Projeto

O **Formação Sem Barreiras** é uma plataforma web totalmente funcional voltada para capacitação profissional e preparação para entrevistas, com foco em acessibilidade digital.

O projeto conta com recursos inclusivos, menu de acessibilidade e integração com banco de dados utilizando Firebase.

---

## 👩‍💻 Minha Participação no Projeto

Neste projeto, fui responsável por:

- Desenvolvimento da página **Admin**
- Desenvolvimento da página de **Histórico**
- Integração completa do sistema com o **Firebase**
- Conexão entre páginas e banco de dados
- Organização da lógica de funcionamento do sistema
